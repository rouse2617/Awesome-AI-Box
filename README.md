 

# 🧰 Awesome-AI-Box

> 收集并整理日常工作流中那些**简单、好用、高效**的 AI 管理与应用工具。

## 📖 简介

在这个 AI 爆发的时代，寻找趁手的工具比盲目尝试更重要。这个仓库是我个人的“AI 工具箱”，记录了我亲测好用的 AI 客户端、模型管理工具、提示词优化器等。

---

## 🛠 工具分类

### 1. 🤖 AI 综合客户端 (Chat Clients)

*用于连接各种大模型 API 的桌面端或 Web 端工具。*

### 2. 🏗️ 本地模型管理 (Local LLM)

*在本地运行和部署开源模型的利器。*

### 3. ✍️ 提示词与生产力 (Prompt & Productivity)

* **UI UX Pro Max Skill** 结合 Cursor/Cline 使用，能显著提升生成界面的美观度，无需冗长的描述即可获得高质量 UI。
